Kesimpulan dan hasil latihan saya pada pertemuan ke-sembilan

  1. Untuk melihat ukuran yang digunakan pada photoshop bisa tekan f12 pada halaman web.
  2. Cara melihat tinggi dokumennya dapat dengan cara menjumlahkan uk. header dan footer
  3. Membuat header dengan Rectangular Marquee Tool, atur widht dan heigtnya, kemudian letakan ditempat yg kita inginkan dan berikan warna
  4. Membuat bagian navigasi dengan cara yang sama menggunakan Rectangular Marquee Tool.
  5. Membuat bagian konten dengan mengcopypaste bagian navigasi, merubah nama, kemudian menyesuaikan ukuran dengan cara ctrl+T.
  6. Membuat menu pada bagian navigasi dengan cara Rounded rectangle tool.
  7. Cara slicing desain dan save web, pilih slice tool kemudian pilih bagian bagian pada layer.
  8. Kemudian save for web, denga cara klik file kemudian pilih save for web, ubah menjadi png kemudian ubah image only menjadi html and image
