Kesimpulan dan hasil latihan saya pada pertemuan ke-sepuluh

  1. Optimalisasi dilakukan untuk menjadikan uk. file menjadi lebih kecil.
  2. Yang dapat membuat web lebih cepat di akses atau dibuka oleh pengguna.
  3. Semakin besar Uk. File semakin lama untuk me-load web.
  4. Cara pertama dengan membuka halaman web, lalu masuk ke inspect Element/Developer Mode : ctrl+shift+i.
  5. Klik bagian network, lalu tekan ctrl+r untuk melihat file yg digunakan pada web.
  6. Kemudian buka pada folder images pada folder kerja, cari gambar sesuai judul pada network, lalu buka dengan photoshop.
  7. Kemudian lakukan proses crop sesuai dengan petunjuk. Lalu save for web dengan judul baru dan format images only
  8. Kemudian buka semua file dokumen di notepad++ untuk merubah codingan. Seperti home_01.png diganti dengan header.png
  9. Cara mengganti semu codingan yang dinginkan pada semua dokumen yaitu ctrl+h lalu cari nama dan ganti sesuai catatan diatas.
  10. Setelah semua diganti, kembali ke halaman web lalu lihat maka terjadi pengurangan uk. file yang drastis
  11. Lakukan lagi pada bagian-bagian lainnya yang dapat dilakukan seperti footer atau background.
  12. Jika sudah merubah yang lain makan uk. file akan turun yang tadinya 59.1 kb menjadi 15.5 kb
  13. Hasil Optimalsisasi, terjadi pengurangan ukuran web.(home.html)
